Suppose you have a very large dataset

res<-rnorm(n=1000,mean=0,sd=1)

and you wish to save it somewhere in an organized way. Then use the save RDS file addin to insert the saveRDS code.

The addin automatically create the directory and the script for saving the file.

saveRDS(res,
    file = "../../Rds_files/2018-07-02-saving-rds-files/res.RDS")

To load the file, use the Read RDS file addin

res<-readRDS(
    file = "../../Rds_files/2018-07-02-saving-rds-files/res.RDS")
